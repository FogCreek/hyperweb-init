# HyperWeb init

... Inits a HyperWeb with Express and file support


Init ✨
-------

```js
var hw = require('hyperweb-init');
app = hw.init();
```

Usage 🐙
---------

use `app` like you would in [express](http://expressjs.com/en/starter/basic-routing.html)
